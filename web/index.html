Market Data Stream Normalizer

Project summary

Server ports

  • feed ingest port: 9000
  • subscriber port: 9100
  • admin http port: 9200

How to run

bash scripts/build.sh
./build/ingest_server
./build/feed_generator 1
./build/subscriber_client

Protocol summary

All messages framed with 4 byte big endian length then 1 byte type then payload
type 0x01: feed tick binary
type 0x10: subscribe request json
